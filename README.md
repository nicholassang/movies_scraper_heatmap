
# Popular Movies Filming Locations

Ever wondered where on earth movies film their scenes ? 

This website shows the top 100 currently most popular movies by IMBD, kept current by a webscraping ETL pipeline with a cron job that runs every 6 hours. 

## Technical

- Frontend: React + Typescript + globe.gl
- Backend: Python FastAPI
- Web Scraping: Selenium
- Cron Job: apscheduler
- Hosted on: AWS Lightsail + Docker Compose
